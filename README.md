# SelektorDisc

Легкий аналог Discord для видео звонков с комнатами, чатом и WebRTC.

## Что это

`SelektorDisc` - это мой проект на Go, в котором реализован базовый realtime-сервис:
- регистрация и логин пользователей;
- список комнат и создание комнаты с именем;
- подключение участников в комнату по WebRTC;
- текстовый чат в комнате;
- сохранение истории сообщений в PostgreSQL;
- звуковые стикеры;
- работа через TURN;

## Текущий функционал

- Auth на cookie-токенах.
- WebSocket для:
  - сигналинга WebRTC;
  - чата;
  - счетчика участников.
- Комнаты хранятся в БД.
- История чата подгружается при входе.
- Ограничение битрейта/разрешения видео на клиенте.
- Поддержка прод-режима с внешним TURN.

## Стек

- **Backend:** Go, Fiber, Pion WebRTC
- **Frontend:** HTML/CSS/Vanilla JS
- **DB:** PostgreSQL
- **Infra:** Docker / Docker Compose

## Структура проекта

```text
cmd/                    # точка входа
internal/
  handlers/             # HTTP/WebSocket handlers
  middleware/           # auth middleware
  repository/           # доступ к БД
  server/               # инициализация приложения
pkg/
  webrtc/               # SFU-логика и peer management
  chat/                 # chat hub
assets/                 # JS/CSS/звуки
views/                  # шаблоны страниц
migrations/             # SQL-миграции
containers/composes/    # docker-compose окружения
tools/turn/             # standalone TURN utility (опционально)
```
